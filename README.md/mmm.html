Марғау

Марғау/Мысық - өте саққұлақ жыртқыш. Қорегін аулағанда дыбысын сездірмей, саусақтарының ұшындағы тырнақтарын майлы табанына жасырып алып, білдіртпей жүреді.

Больше информации о карточке 1.

Күшік

Күшік – иттің және ит тұқымдас жыртқыш аңдардың баласы. Күшіктер аңқау болып келеді, бірақ та өте көңілді және мейірімді жануарлар.

Больше информации о карточке 2.

Хомяк

Хомяк (лат. Cricetinae) - кеміргіштер тұқымдасына жататын хомяктар тұқымдасының кіші тұқымдасы. Жеті туысқа жататын 19 түрді қамтиды. Үй жануарлары ретінде танымал болды
